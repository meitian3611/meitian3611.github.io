vue文本格式化

发表于 2021-03-19 更新于 2022-02-28 分类于 web前端 阅读次数：

vue-text-format

一个功能强大的 vue 自定义文本格式扩展插件。

vue-text-format这个扩展移植了excel的功能，可以在页面上对数据改变显示格式。 使用方式也很简单，通过扩展vue的自定义命令v-format的形式，绑定格式，就可以将内部的文本进行转换。

使用方法

安装

  • npm install vue-text-format

    在main.js入口文件引用插件

使用

  • 方式1：指令v-format 通过v-format传入想要转换的格式
  • 方式2：函数调用，vue原型上绑定了一个textFormat函数

示例

  • 我们通过几个例子，让你了解一下这个格式代码功能到底有多全多强大

常用格式编码介绍

数字占位符

  • ‘#’： 只显有意义的零而不显示无意义的零。小数点后数字如大于”#”的数量，则按”#”的位数四舍五入。
  • ‘0’： 如果单元格的内容大于占位符，则显示实际数字，如果小于点位符的数量，则用0补足。
代码 数字 显示
###.## 12.1 12.1
###.## 12.1263 12.13
00000 1234567 1234567
00000 123 00123
00.000 100.14 100.140
00.000 1.1 01.100

小数点

代码 文本 显示
0.# 11.23 11.2
0.# 11 11
0.00 3 3.00
0.## 3 3
0.00% 3 300.00%
0.##% 3 300%

百分比

  • “%”：百分比。 例：代码“#%”。“0.1”显示为“10%”
代码 数字 显示
#% 0.1 10%
#.#% 0.131 13.1%
#.##% 0.131 13.1%
#.#0% 0.131 13.10%

千分位分隔符

代码 文本 显示
#,###0.00 12000 12,000.00
  • 如在代码中“,”之后为空，则把原来的数字缩小1000倍。
代码 文本 显示
#, 10000 10
#,”k” 123123 123k
#,, 1000000 1
0,.# 12345 12.3
#.00, 12345 12.35
“人民币 “#,##0,,”百万” 1234567890 人民币 1,235百万

代码 文本 显示
YYYY-MM-DD 1562838244 2019-07-11

HH:mm:ss 1562838244 17:44:04
